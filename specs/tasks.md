# Superlista — Plan de trabajo

## Épica 1: Fundaciones del proyecto
- Crear repositorio.
- Configurar entorno virtual Python 3.12.12.
- Configurar Django y MySQL.
- Crear apps base.
- Configurar Bootstrap 5.3.8.

## Épica 2: Autenticación y perfiles
- Registro de usuario.
- Login y logout.
- Recuperación de contraseña.
- Perfil básico.

## Épica 3: Listas colaborativas
- Crear lista.
- Editar lista.
- Archivar lista.
- Generar token de compartición.
- Invitar por correo.

## Épica 4: Permisos y membresías
- Modelo ListMember.
- Roles owner/editor/viewer.
- Validación por acción.
- Revocación de acceso.

## Épica 5: Ítems y categorías
- CRUD de ítems.
- CRUD de categorías.
- Estado de ítems.
- Selector dinámico de productos frecuentes.

## Épica 6: Proveedores
- CRUD de proveedores.
- Asociación de listas y compras por proveedor.
- Dirección y tipo de proveedor.

## Épica 7: Compras y boletas
- Registrar compra.
- Agregar líneas de compra.
- Cargar precio efectivo.
- Subir boleta.
- Comparar precio estimado vs real.

## Épica 8: Reportes
- Totales por lista.
- Totales por proveedor.
- Totales por período.
- Resumen de diferencias.

## Épica 9: Seguridad y auditoría
- Auditoría de cambios.
- Validaciones de acceso.
- Tokens revocables.
- Protección de archivos.

## Épica 10: Calidad y despliegue
- Pruebas unitarias.
- Pruebas de integración.
- Configuración de producción.
- Variables de entorno.
- Documentación técnica final.
